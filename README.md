## Current Version
**Version:** Beta 1.0

### Included in this build:
- Core gameplay mechanics and board logic
- Single-player mode against Stockfish-powered AI (Opponent and Neutral AI)
- AI configurations with difficulty adjustments
- Fully themed UI and sound effects

### Work In Progress (WIP) - Disabled/Coming Soon:
- **Save Game functionality**: The ability to save/load mid-game matches is currently disabled as we are refining the format to maintain compatibility across versions.
- **Network Gameplay**: Multiplayer matchmaking and online play are currently actively in development and will be available in a future update. The Online Lobby is included but currently restricted.

## Feedback and Bug Reports
Since this is a Beta build, you might encounter unexpected behaviors or bugs.

**How to provide feedback:**
We are collecting all feedback, bug reports, and suggestions through our [itch.io project page](https://itch.io).
- Please leave a comment with any bugs you find. Be as detailed as possible!
- If you have suggestions for improvements, balance changes, or features you'd like to see, we would love to hear them!

Thank you again for playing King's Trial. We hope you enjoy testing it out as much as we enjoy making it!
